


# Mobile Camera Streaming System

A simple web-based system that allows a mobile phone camera to stream live images to a laptop or any device browser in real-time. The project is built using **Python (Flask)** and **HTML5**, with optional **ngrok** tunneling for remote access.

---

## Features

- Capture mobile phone camera images every 2â€“5 seconds  
- Live feed displayed on laptop or any browser  
- Single-device session lock for secure streaming  
- Works remotely using ngrok without installing apps on the phone  

---

## Tech Stack

- Python (Flask)  
- HTML / CSS / JavaScript  
- Ngrok (optional, for remote access)
